Техническое задание

Страница Articles:

На странице отображается список статей. Статьи отсортированы по дате. Если статей более 10, то должен осуществляться постраничный вывод. При нажатии на заголовок статьи происходит переход на страницу статьи.
Для пользователя с привилегией Sale:
- Кнопка " Add Article " с переходом на страницу "Add Article".
- Возле статей разместить Checkbox, который отмечает статьи, для удаления. При нажатии на кнопку «Delete Article» - удаляются все статьи, отмеченные чекбоксом.

Страница Article:

На странице отображается заголовок статьи и текст статьи и комментарии. Если комментариев больше 10, то должен осуществляться постраничный вывод.
Для пользователя с привилегией Sale:
- Кнопка  " Change Article" с переходом на страницу " Change Article ".
- Возле каждого комментария есть кнопка "Delete" позволяющая удалить комментарий.
Для пользователя с привилегией Customer: • Под текстом статьи имеется окно для ввода текста комментария и кнопкой "Add comment".

Страница Add Article:

Страница доступна пользователю с привилегией Sale. На странице имеются поля:
- Title, Text of article, кнопка "Add Article " для добавления статьи.

Страница Change Article:

Страница доступна только пользователю с привилегией Sale. На странице имеются поля: Title, Text, кнопоки: "Save", "Cancel" с возвращением к списку статей без изменений.

Страница Items:

На странице отображается список товаров. Товары должны быть отсортированы по названию в алфавитном порядке. Если товаров больше 10, то осуществляется постраничный вывод.
Для пользователя с привилегией Sale:
На странице есть кнопки "Add Items" и "Upload Items".  Для каждого товара есть кнопки "Update", "Copy" при нажатии на которую происходит переход на страницу "Copy Items", Checkbox и кнопка "Delete", при нажатии на которую удаляются отмеченные товары. Отображаются все отмеченные товары на удаление.
Для пользователей с привилегией Customer:
- У каждого товара в форме есть поле с количеством и кнопка "Order”.

Страница Update Items:

Страница доступна только для пользователя с привилегией Sale. На странице присутствует таблица с  полями: Title, Price, Unique number; кнопки: "Save”,  "Cancel", возвращает на страницу "Items" без внесения изменений.

Страница Copy Items:

Страница доступна только для пользователя с привилегией Sale. На странице присутствует таблица с полями: Title, Price, Unique number; кнопки: "Save", "Cancel", возвращает на страницу "Items" без сохранения.

Страница Add Item:

Страница доступна только для пользователя с привилегией Sale. На странице присутствует таблица с полями: Title, Price, Unique number; кнопки: "Add", "Cancel", возвращает на страницу "Items" без сохранения.

Страница Upload Items:

Страница доступна только для пользователя с привилегией Sale. На странице присутствует форма для загрузки файла с товарами.

Страница Orders:

На странице должен быть отображен список заказов. Заказы отсортированы по дате. Отображаются товары с определенным статусом. Если заказов больше 10, то осуществляется постраничный вывод. Для пользователя с привилегией Sale:
Отображаются заказы всех пользователей.
Возможность обновить статус заказа, кнопка "Update" в форме заказа.
Для пользователей с привилегией Customer: Отображаются только их заказы.

Страница Profile:

Страница доступна только для пользователя с привилегией Customer. На странице отображаются поля: Login, Lastname, Name, Phone number, Address.
Кнопки: "Change" с переходом на страницу "Change information", "Change Password" с переходом на страницу "Change Password".

Страница Change Information:

Страница доступна только для пользователя с привилегией Customer. На странице отображаются поля: Lastname, Name, Phone number, Address; кнопки: "Save", "Cancel", с переходом на страницу "Profile" без сохранения изменений.

Страница Change Password:

Страница доступна только для пользователя с привилегией Customer. На странице отображаются поля: Old password, New password, Repeat password.
